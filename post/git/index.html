Git常见操作

2019-10-26

创建分支

git checkout -b dev
git push origin dev
git branch --set-upstream-to=origin/dev 设置默认

拉取分支代码

git clone -b d dev 【链接】

当前分支落后于主分支

根据主分支（origin）代码在本地创建新的临时分支，命名为temp

git checkout -b temp origin

将临时分支（temp）合并到开发分支（dev）

git checkout dev
git merge temp

解决合并后的冲突
提交开发分支（dev）并push到远程仓库
后将开发分支（dev）合并到主分支（origin）
删除本地分支（temp）

git branch -d temp

撤销合并

git log
git reset --hard 【版本号】
